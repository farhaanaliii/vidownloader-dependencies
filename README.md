# ViDownloader Dependencies

Binary dependencies for [ViDownloader](https://github.com/farhaanaliii/ViDownloader-v2) - A powerful video downloader application.

## About

This repository hosts binary dependencies required for ViDownloader to function properly. These files are distributed via GitHub Releases to provide fast, reliable downloads worldwide.
